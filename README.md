# Angular - The Complete Guide

## About

This repository contains the projects from the "Angular - The Complete Guide" course by Maximilian Schwarzmüller on Udemy. The course covers how to build modern web applications using Angular by making projects. I will be updating this repository with my progress as I work through the course.
